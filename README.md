## 项目概述

本项目基于 MuJoCo MPC 仿真框架，实现了一个汽车仪表盘可视化系统。系统能够实时获取车辆仿真数据，包括速度、转速、油量和温度等关键参数，并通过 OpenGL 绘制指针式和数字化仪表盘进行可视化显示。仪表盘以 HUD 方式覆盖在三维仿真场景上，实现数据与画面的实时同步，为用户直观呈现车辆运行状态。项目在数据获取、界面渲染和实时更新方面完成了完整功能，同时实现了部分 UI 动画效果和警告提示。

## 功能实现

### 已实现功能详细说明

1. **速度表**
   速度表通过从 MuJoCo 仿真系统中实时读取车辆的线速度数据进行计算和更新。系统将获取到的速度值转换为符合实际驾驶习惯的单位，并利用 OpenGL 绘制指针式或刻度式速度表界面。随着车辆在仿真环境中的运动变化，速度表能够实时刷新，直观反映车辆当前的行驶状态，为用户提供清晰的速度信息反馈。
2. **转速表**
   转速表用于显示发动机当前转速情况。程序从仿真模型中读取发动机或动力系统相关的状态参数，并映射到转速表的角度变化。通过指针旋转和刻度标识，转速表能够反映发动机负载变化和工作区间，使用户可以直观了解车辆动力系统的运行状态。
3. **数字显示（油量、温度）**
   系统以数字化方式显示车辆的油量和温度等信息。相关数据从仿真模型中实时获取，并在界面上以文本形式呈现。相比指针式仪表，数字显示具有读数准确、信息清晰的特点，能够辅助用户快速掌握车辆的剩余油量和当前工作温度情况。
4. **小地图（未实现）**
   小地图功能原计划用于显示车辆在仿真场景中的实时位置和运动轨迹，通过二维视图直观呈现车辆行驶路线。然而由于时间和开发精力限制，该功能尚未完成，仅在设计阶段进行了初步构想，后续可作为系统扩展功能继续完善。

## 文件说明与解释

- **`dashboard_data.h`（数据获取模块）**
  该文件主要负责从 MuJoCo 仿真系统中读取车辆运行相关的数据，包括速度、转速、油量、温度等状态信息。模块对底层仿真数据进行封装与处理，为上层可视化模块提供统一、稳定的数据接口，降低各功能模块之间的耦合度。
- **`dashboard_render.h`（渲染模块）**
  该文件主要实现仪表盘界面的图形绘制功能。通过 OpenGL 渲染接口，将数据获取模块提供的实时车辆状态以速度表、转速表以及数字信息等形式显示在界面上，并以 HUD 的方式叠加到 MuJoCo 的三维仿真画面中，实现直观的可视化效果。
